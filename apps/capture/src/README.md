# Source
Front-End de la aplicacion capture.

## Components
en `components/` se encuentran los componentes reutilizables de la aplicacion.

## Hooks
en `hooks/` se encuentran los hooks personalizados de react utilizados en la aplicacion.

## Lib
en `lib/` se encuentran las funciones y utilidades generales que no pertenecen a un componente o hook especifico.

## Types
en `types/` se encuentran las definiciones de tipos TypeScript utilizadas en la aplicacion
